# Water Quality Prediction

This project focuses on predicting water quality using data analysis and machine learning techniques.

## Objective

To determine whether water is safe or unsafe based on various chemical and physical parameters.

## Features

* Data collection and preprocessing
* Exploratory Data Analysis (EDA)
* Model building and evaluation

## Tools & Technologies

* Python
* Jupyter Notebook
* Pandas, NumPy, Scikit-learn

## Results

The model predicts water quality with reasonable accuracy based on input parameters.

* Improve model accuracy
